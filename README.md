**Basic Response Data**

| 변수이름   | 자료형      | 설명                                     | 예시        |
| ---------- | ----------- | ---------------------------------------- | ----------- |
| RES_STATUS | number      | http 상태코드입니다.                     | 200         |
| RES_CODE   | string      | 응답성공하면 'success' 아닐시 오류메시지 | 'success'   |
| RES_DATA   | objets/list | 각 함수에 필요한 Data                    | 각자 데이터 |
